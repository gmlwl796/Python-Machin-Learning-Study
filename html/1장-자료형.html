1.7.8 요약 및 정리¶
  • 분류문제에서 각 품종을 '클래스'라고 정의

  • 개별 붓꽃의 품종은 '레이블'

  • 붓꽃 데이터셋은 두 개의 Numpy 배열로 이루어짐

  • X : 데이터를 담고 있음, 2차원 배열(각 데이터 포인트는 행. 각 특성은 열)

  • y : 정확한 혹은 기대하는 출력을 갖음, 1차원 배열(각 샘플의 클래스 레이블에 해당하는 0 ~ 2값으로 구성)

  • 모델구축에 사용할 훈련데이터 X_train, y_train

  • 새로운 데이터에 얼마나 잘 적용할 수 있을지 평가하기 위한 테스트 세트 X_test, y_test

  • KNN : 새 데이터 포인트를 예측하기 위해 훈련 데이터에서 가장 가까운 이웃 선택

  • KNeighborsClassifier 클래스 -> 구현, 모델, 예측 가능
